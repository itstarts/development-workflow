# AGENTS.md — tests

## 适用范围

适用于仓库级契约测试和跨 skill 集成测试。

## 规则

- 测试先于生产行为修改；RED 必须因目标缺失或错误行为失败，而不是因测试本身错误。
- 测试使用临时目录和虚构路径，不读写真实用户 skill、task/thread 或生产状态。
- 固定判据与原始输出分离；代理不得看到 expected 文件。
- 不通过降低断言、跳过失败场景或静默兜底使测试变绿。
- 测试必须可离线重复运行，不依赖 GitHub、插件缓存版本或用户凭证。
- 跨 skill 测试验证文档路径、批准状态和显式交接字段，不直接调用兄弟 skill 内部实现。
