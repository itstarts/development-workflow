# 六段旁白素材

## 1. 清晨的城市

天刚亮，街道先从送货车、清洁车和早餐铺的蒸汽中醒来。城市并不是被闹钟同时叫醒，而是一层一层恢复声音。

## 2. 通勤的人

地铁入口很快聚起人流。有人检查当天的会议，有人戴着耳机，还有人只是安静看着车窗里的倒影。

## 3. 午间短暂停顿

中午，写字楼之间出现短暂松动。长椅、便利店和小餐馆承接了半小时的休息，也让陌生人的路线偶尔交叉。

## 4. 午后的工作

阳光越过楼面后，办公室重新变得紧凑。键盘、电话和打印机组成重复节奏，细小决定在其中不断发生。

## 5. 黄昏转换

傍晚的光线让玻璃幕墙显出另一种颜色。下班的人流与刚开始营业的店铺相遇，城市从工作模式转向生活模式。

## 6. 夜晚仍在继续

深夜并不意味着完全安静。便利店、急诊室、夜班公交和仍亮着的窗口提醒我们：一座城市总有人在接住下一段时间。
